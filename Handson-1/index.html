What is CSS?

  1. CSS Stands for Cascading Style Sheets
  2. CSS describes how HTML elements are to be on screen, paper, or in other media.
  3. Css is not necessery CSS saves a lot of work. It can be control the layout of multiple web papes all at once
  4. External stylesheets are stored in CSS file

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML element |
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies astyle to multiple elements. ID, on the other hand, applies a style to one unipue element. ID is also spacial in that you can use a spacial URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a spacific element or range of elements on a web page. Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to targer elements to which a style should be applied.
